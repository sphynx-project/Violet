# Violet

A simple and light bootloader that aims to support multiple boot protocols.

## Supported Targets

Violet only supports x86_64 (UEFI) as of now. 

## Supported Protocols

| **Protocol** | **Supported** | **Arches**  |
|--------------|---------------|-------------|
| sphynxboot   | ❌, ❓        | x86_64      |

- ❓: Work in progress
- ✅: Done
- ❌: Not Done
